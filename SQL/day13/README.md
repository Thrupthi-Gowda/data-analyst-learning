
# Day 13 – SQL FOREIGN KEY

## Overview
On Day 13, I learned how to create relationships between tables using FOREIGN KEY.

## Topics Covered
- FOREIGN KEY
- Table relationships
- Data integrity

## Description
A foreign key links one table to another, ensuring that only valid data is inserted.

## Key Learnings
- FOREIGN KEY connects two tables.
- It prevents invalid data insertion.
- It maintains data consistency.
