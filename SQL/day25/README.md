# Day 25 – SQL CASE WHEN

## Overview
Learned how to apply conditional logic in SQL.

## Topics Covered
- CASE WHEN
- Conditional expressions
- Categorizing data

## Key Learnings
- CASE works like IF-ELSE
- Useful for classification and business logic
- Can be combined with GROUP BY

## Example
CASE 
  WHEN salary > 50000 THEN 'High'
  ELSE 'Low'
END

- MySQL
