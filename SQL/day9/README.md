# Day 9 – SQL UPDATE and DELETE

## Overview
On Day 9, I learned how to modify and remove data from a table using SQL UPDATE and DELETE statements.

## Topics Covered
- UPDATE statement
- DELETE statement
- WHERE clause importance

## Description
UPDATE is used to modify existing records, while DELETE is used to remove records from a table.

## Key Learnings
- UPDATE changes existing data.
- DELETE removes records permanently.
- WHERE clause is very important to avoid affecting all rows.
