# Day 24 – Warehouse Inventory Management

## Overview
This project simulates a warehouse inventory system using SQL.

## Features
- Manage warehouses and products
- Track product quantity and price
- Handle orders

## Concepts Used
- CREATE, INSERT, SELECT
- WHERE, UPDATE, DELETE
- JOIN
- GROUP BY, HAVING
- ORDER BY, LIMIT
- SUBQUERY
- WINDOW FUNCTIONS

## Key Insights
- Identify low stock products
- Analyze warehouse distribution
- Rank products by price

## Tools Used
- MySQL
- Git
- GitHub
