# Day 10 – SQL ORDER BY and LIMIT

## Overview
On Day 10, I learned how to sort and limit data using SQL ORDER BY and LIMIT clauses.

## Topics Covered
- ORDER BY
- ASC (Ascending)
- DESC (Descending)
- LIMIT

## Description
ORDER BY is used to sort data, and LIMIT is used to restrict the number of rows returned.

## Key Learnings
- ORDER BY helps arrange data in ascending or descending order.
- LIMIT is useful to get top or bottom records.
- These are commonly used in reporting and dashboards.
