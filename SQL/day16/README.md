# Day 16 – SQL GROUP BY

## Overview
On Day 16, I learned how to group data and perform aggregate functions.

## Topics Covered
- GROUP BY
- COUNT()
- AVG()
- MAX()
- MIN()

## Key Learnings
- GROUP BY groups rows with same values.
- COUNT is used to count rows.
- AVG calculates average.
- MAX finds highest value.
- MIN finds lowest value.

## Practice Includes
- Counting employees
- Grouping by department
- Salary analysis
