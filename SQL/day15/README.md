# Day 15 – SQL Practice (Joins + Queries)

## Overview
On Day 15, I practiced real-world SQL queries using JOIN, WHERE, and ORDER BY.

## Topics Covered
- INNER JOIN
- LEFT JOIN
- WHERE condition
- ORDER BY

## Key Learnings
- INNER JOIN returns matching data.
- LEFT JOIN returns all records from left table.
- WHERE is used to filter data.
- ORDER BY is used to sort results.

## Practice Includes
- Joining two tables
- Filtering employees by salary
- Sorting results
- Finding employees by department
