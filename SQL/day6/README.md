# Day 6 – Advanced SQL Joins

## Overview
On Day 6, I practiced advanced SQL queries by combining JOIN operations with filtering, grouping, and aggregation.

## Topics Covered
- JOIN with WHERE
- Aliases (AS)
- GROUP BY with JOIN
- HAVING clause
- Filtering after JOIN

## Key Learnings
- JOIN can be combined with conditions to filter data.
- Aliases make queries shorter and readable.
- GROUP BY helps summarize joined data.
- HAVING is used to filter aggregated results.
