# Project 1 – Employee Data Analysis

## Overview
This project analyzes employee data using SQL.

## Objectives
- Find highest salary
- Calculate average salary
- Identify top employees
- Analyze department performance

## Key Queries
- MAX, AVG, SUM
- GROUP BY
- ORDER BY
- Subqueries

## Insights
- IT department has high salary distribution
- Finance department shows highest average salary
- Top employees identified based on salary

- MySQL
- Git
- GitHub
