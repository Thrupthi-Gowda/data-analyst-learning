# Day 7 – SQL Subqueries

## Overview
On Day 7, I learned how to use subqueries to perform complex queries by nesting one query inside another.

## Topics Covered
- Subquery in WHERE clause
- Subquery with IN
- Subquery with AVG, MAX, MIN

## Key Learnings
- Subqueries allow dynamic filtering based on other query results.
- They are useful for comparisons like above average or highest value.
- Commonly used in real-world data analysis.
