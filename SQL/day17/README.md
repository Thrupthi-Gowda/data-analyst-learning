# Day 17 – SQL HAVING

## Overview
On Day 17, I learned how to filter grouped data using HAVING.

## Topics Covered
- GROUP BY
- HAVING
- Aggregate filtering

## Key Learnings
- WHERE filters rows before grouping.
- HAVING filters groups after grouping.
- Used with aggregate functions like AVG, COUNT.

## Practice Includes
- Average salary filtering
- Counting employees per department
