# Day 12 – SQL ALTER TABLE

## Overview
On Day 12, I learned how to modify an existing table using ALTER TABLE.

## Topics Covered
- ADD column
- MODIFY column
- CHANGE (rename column)
- DROP column

## Description
ALTER TABLE is used to change the structure of an existing table without deleting it.

## Key Learnings
- ADD is used to insert new columns.
- MODIFY changes datatype of a column.
- CHANGE renames a column.
- DROP removes a column.

- MySQL
