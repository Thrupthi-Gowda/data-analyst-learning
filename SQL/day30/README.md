📌 Description

On Day 30, I focused on Window Functions, which are essential for solving real-world data problems and are commonly asked in SQL interviews.

Unlike GROUP BY, window functions allow performing calculations across rows without collapsing the dataset.

---

🧠 Key Concepts

🔹 ROW_NUMBER()

Assigns unique row numbers to each row.

🔹 RANK()

Assigns same rank for duplicate values but skips next rank.

🔹 DENSE_RANK()

Assigns same rank without skipping numbers.

🔹 PARTITION BY

Used to divide data into groups while keeping all rows.

---

💼 Real Use Cases

- Finding top N records
- Ranking employees by salary
- Department-wise analysis
- Removing duplicates
- Identifying second highest values
